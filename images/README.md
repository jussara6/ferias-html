Coloque aqui a sua foto para o Hero com o nome `jussara-hero.jpg` (ou .webp).

Recomendações:
- Tamanho ideal: 1200x1200 px para maior qualidade; o CSS exibirá 260x260 (desktop) e 180x180 (mobile).
- Formato: prefira WebP ou JPG otimizado.
- Nome do arquivo: `jussara-hero.jpg` ou `jussara-hero.webp` (substituir o .svg atual).
- Otimize a imagem antes de subir (Squoosh, TinyPNG, etc.).

Exemplo de comando (Linux/macOS) para redimensionar com ImageMagick:

    magick convert input.jpg -resize 1200x1200^ -gravity center -extent 1200x1200 jussara-hero.jpg

Depois de adicionar sua foto, atualize `images/jussara-hero.jpg` e o site mostrará sua foto no Hero.
